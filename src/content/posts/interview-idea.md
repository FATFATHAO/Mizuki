# 凡事预则立，不预则废

前言: 这个星期在仓促之间报名了字节工程训练营的笔试，原本是学校大三不让出去日常实习，所以想找个线上的实习，如果~~能延续到寒假~~ 那就更好了。字节这个训练营感觉就挺适合我的，又因为大一大二因为参与了学校的实验室，学会并且使用了 React 做了几个项目，感觉自己的能力应该也能找到一个实习，遂报名。

可惜自己~实力不行~(其实是`名词王、半桶水`)，选择题关于 408 的内容倒是没问题，反而是自己经常触碰的，比较喜欢的 coding 部分出错的比较多，关于一些基础的 JavaScript 知识都做的我很痛苦。这个还是只有笔试的训练营测试，很难想象如果真的要去找实习，相关的八股知识以及面对面交谈的时候~可能直接~(`并非可能`)就被面试官拿下了。所以不但要现在就开始看一些面经，同时还要对基础的知识进行梳理。我本身认为我的 cpp 能力和前端能力是最强的，经过了这次笔试之后，我发现实际上是 cpp 的能力更强，而前端的能力和理论知识都有所疏忽，这是不行的！必须要寻求改变!!!所以从今以后，我会开始在博客梳理相关的技术知识，同时也是为了帮助自己可以更好的复习，梳理知识！

# 求稳第一，但也不要太过于纠结

在进行测试的时候，我发现我对于不确定的题目会反复的检查和思考(甚至可能思考了之后根本也想不到正确答案，甚至依旧才求证我的答案会不会是正确答案，和三年前的高考一样。这本质是我自己心理没有底气，又知道这件事情的重要导致的)这种情况说白了还是因为自己理论和技术知识不到位，我不会简单的认为我的技术就可以了，实际上这些都不到位才会引发的一系列这样的问题，难道不对吗？

所以我认为，想要改变现状，一个就是从现在开始进入理论梳理和复习的状态，另一个就是一定要保持刷题的手感，leetcode hot100 刷了吗？自己对于很多算法知识真的会了吗？这应该是一个不时要回答自己的问题。最近在做一个《算法分析与设计》的大作业，题目是关于五子棋 AI 的，之后有机会也要梳理知识放到博客上，不然不就白做了吗？我想用一句话来激励自己，又不希望这句话太过于直白，但自己的文学水平又不高，干脆就用：“光阴不与闲人浪，须把精神注有方”来激励自己吧。

### 希望以后真的有机会到大厂实习，转正工作吧

再多的后悔和懊恼，都不如从现在开始抛弃幻想脚踏实地的开始奋斗，也就是人们常说的：“抛弃幻想，准备斗争”了吧
