# 凡事预则立，不预则废

前言: 这个星期在仓促之间报名了字节工程训练营的笔试，原本是学校大三不让出去日常实习，所以想找个线上的实习，如果~~能延续到寒假~~ 那就更好了。字节这个训练营感觉就挺适合我的，又因为大一大二因为参与了学校的实验室，学会并且使用了 React 做了几个项目，感觉自己的能力应该也能找到一个实习，遂报名。

可惜自己~实力不行~(其实是`名词王、半桶水`)，选择题关于 408 的内容倒是没问题，反而是经常 coding 的部分出错的比较多，关于一些基础的 JavaScript 知识都做的我很痛苦。这个还是只有笔试的训练营测试，很难想象如果真的要去找实习，相关的八股知识以及面对面交谈的时候~可能直接~(`并非可能`)就被面试官拿下了。所以不但要现在就开始看一些面经，同时还要对基础的知识进行梳理。我本身认为我的 cpp 能力和前端能力是最强的，经过了这次笔试之后，我发现实际上是 cpp 的能力更强，而前端的能力和理论知识都有所疏忽，这是不行的！所以自从今天之后，我会经常在博客梳理相关的技术知识，同时也是为了帮助自己可以更好的复习，梳理一下知识！
